Start Match:

  1. Click the Start Game button.
  2. Click anywhere to start playing.
  3. Click the cards to match them up.
Back

Game Rules:

  • You must pair two cards that contain the same person.
  • You cannot pair nonmatching cards ingame, stop it please.
  • You must match all cards before the timer runs out, otherwise resulting to a loss.

Additional Features:

1. Timer - By booting up the game will start the countdown until it reaches zero, which ends the game

2. Flips - By clicking the cards will be recorded by this system. Although, it doesn't affect your score, you can try to be competitive with it.
